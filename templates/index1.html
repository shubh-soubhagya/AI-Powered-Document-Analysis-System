Processing Information
Why does processing take time?

Document Buddy performs several intensive operations:

  • Text Extraction: Extracting text from multiple PDFs
  • Plagiarism Detection: Running multiple similarity algorithms on document pairs
  • Vector Database: Creating embeddings for semantic search
  • Document Classification: Analyzing document content to predict categories

The processing time depends on the number and size of your PDF documents.

About Document Buddy

Document Buddy is an AI-powered document analysis system that helps you:

  • Detect plagiarism between documents using multiple algorithms
  • Classify documents into categories automatically
  • Chat with your documents using advanced AI

Built with Python, Flask, and various NLP technologies.
